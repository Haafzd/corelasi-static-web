# CORELASI Static Web

Prototype static web CORELASI menggunakan HTML, CSS, dan JavaScript.

## Halaman utama
- Landing page
- Sign in
- Dashboard
- Course
- Teaching schedule
- Layout dasar

## Struktur
- `*.html` — halaman static
- `css/` — styling per halaman
- `js/` — script frontend
- `data/` — data JSON dummy
- `aset/` — gambar dan ikon

## Menjalankan
Buka file HTML langsung di browser, atau gunakan static server sederhana.

```bash
npx serve .
```
